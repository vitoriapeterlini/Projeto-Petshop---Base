# PetShop Amigo Fiel
Um projeto web moderno e responsivo para PetShop Amigo Fiel, desenvolvido com foco em boas práticas de desenvolvimento front-end, escalabilidade e experiência do usuário. Ideal para quem busca um ponto de partida sólido para projetos web institucionais.

---

## ✨ Funcionalidades
- **Homepage** com hero section e chamada para ação
- **Seção de Serviços** (banho, tosa,  veterinário, etc)
- **Galeria de imagens** 
- **Formulário de contato** com validação em JavaScript
- **Seção de localização**
- **Menu responsivo** para dispositivos móveis
- **Design limpo e moderno** com CSS  customizado

---

## 🚀 Como executar
1. Clone este repositório
2. Abra a pasta `Projeto-PetShop`no seu editor ou navegador
3. Basta abrir o arquivo `index.html` em seu navegador preferido
